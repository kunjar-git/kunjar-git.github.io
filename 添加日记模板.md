# 📝 添加新日记指南

## 新增功能说明

现在日记采用**独立 HTML 文件**存储，每篇日记是一个单独的文件！

---

## 📁 文件位置

```
github-io-site/
├── posts/
│   ├── posts-index.json    # 日记索引（所有日记的元数据）
│   ├── 2026-03-25-first-post.html
│   ├── 2026-03-20-book-review.html
│   └── 你的新日记.html
```

---

## ✏️ 添加新日记步骤

### 步骤 1: 创建日记 HTML 文件

在 `posts/` 目录创建新文件，例如：`2026-03-26-my-day.html`

复制以下模板：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>日记标题 - 个人日记</title>
    <link rel="stylesheet" href="../css/style.css">
    <link rel="stylesheet" href="../css/journal.css">
    <link href="https://fonts.googleapis.com/css2?family=Noto+Sans+SC:wght@300;400;500;700&display=swap" rel="stylesheet">
</head>
<body>
    <header class="header">
        <nav class="nav">
            <div class="nav-brand">
                <a href="../index.html">🏠 我的空间</a>
            </div>
            <ul class="nav-links">
                <li><a href="../index.html">首页</a></li>
                <li><a href="../journal/index.html">日记</a></li>
                <li><a href="../timeline.html">时间线</a></li>
                <li><a href="../about.html">关于我</a></li>
            </ul>
        </nav>
    </header>

    <main class="container">
        <button class="back-btn" onclick="history.back()">
            ← 返回日记列表
        </button>

        <article class="post-content">
            <div class="post-header">
                <h1>日记标题</h1>
                <div class="post-meta-large">
                    <span>📅 2026 年 03 月 26 日 星期四</span>
                    <div class="post-tags">
                        <span class="post-tag life">生活</span>
                        <span class="post-tag thoughts">思考</span>
                    </div>
                </div>
            </div>
            <div class="post-body">
                <!-- 这里是日记正文 -->
                <h2>正文标题</h2>
                <p>这里是内容...</p>
            </div>
        </article>
    </main>

    <footer class="footer">
        <div class="footer-content">
            <p>&copy; 2026 我的个人空间。Built with ❤️ on GitHub Pages</p>
            <div class="social-links">
                <a href="https://github.com/kunjar-git" target="_blank">GitHub</a>
                <a href="mailto:your@email.com">Email</a>
            </div>
        </div>
    </footer>
</body>
</html>
```

### 步骤 2: 更新索引文件

编辑 `posts/posts-index.json`，在 `posts` 数组**最前面**添加：

```json
{
  "id": "2026-03-26-my-day",
  "title": "日记标题",
  "date": "2026-03-26",
  "tags": ["life", "thoughts"],
  "excerpt": "简短摘要，50 字以内",
  "content": "正文内容（用 \\n 表示换行）"
}
```

### 步骤 3: 推送到 GitHub

```bash
cd /home/admin/openclaw/workspace/github-io-site
git add .
git commit -m "Add: 新日记 - 日记标题"
git push
```

等 1 分钟，网站自动更新！
