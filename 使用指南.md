# 📖 使用指南

## 🔄 清除缓存（重要！）

GitHub Pages 有缓存，修改后可能不会立即显示。用以下方法强制刷新：

### 方法 1: 浏览器快捷键（推荐）
- **Windows/Linux**: `Ctrl + Shift + R` 或 `Ctrl + F5`
- **Mac**: `Cmd + Shift + R`

### 方法 2: 开发者工具
1. 按 `F12` 打开开发者工具
2. 右键点击刷新按钮
3. 选择「清空缓存并硬性重新加载」

### 方法 3: 清除浏览器缓存
- Chrome: 设置 → 隐私和安全 → 清除浏览数据
- 或者访问：`chrome://settings/clearBrowserData`

## ✏️ 添加新日记（3 步搞定）

### 步骤 1: 告诉我内容

直接说：
> "帮我添加一篇日记，标题是 xxx，内容是 xxx"

我会帮你创建所有文件！

### 步骤 2: 确认内容

我会生成：
- ✅ 日记 HTML 文件
- ✅ 更新索引
- ✅ 更新时间线

### 步骤 3: 推送更新

我帮你运行：
```bash
git add .
git commit -m "Add: 新日记"
git push
```

等 1-2 分钟，网站自动更新！

## 🏷️ 标签筛选功能

日记列表页有快速筛选按钮：

| 按钮 | 功能 |
|------|------|
| 全部 | 显示所有日记 |
| 生活 | 只显示生活类日记（绿色标签）|
| 思考 | 只显示思考类日记（粉色标签）|
| 技术 | 只显示技术类日记（青色标签）|
| 旅行 | 只显示旅行类日记（橙色标签）|

**如果筛选不工作：**
1. 按 `F12` 打开控制台
2. 点击筛选按钮
3. 看 Console 有没有错误信息
4. 截图发给我

## 🐛 常见问题

### Q: 日记显示 0 篇
**A:** 
1. 按 `Ctrl + Shift + R` 强制刷新
2. 打开控制台看错误信息
3. 检查 `posts-index.json` 是否存在

### Q: 筛选功能不工作
**A:**
1. 强制刷新页面
2. 打开控制台看错误
3. 检查标签名称是否正确（life/thoughts/tech/travel）

### Q: 修改后网站没更新
**A:**
1. 确认 `git push` 成功
2. 等 1-2 分钟（GitHub 构建需要时间）
3. 强制刷新浏览器缓存

## 📝 日记模板

创建日记 HTML 文件时，复制以下模板：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>日记标题 - 个人日记</title>
    <link rel="stylesheet" href="../css/style.css">
    <link rel="stylesheet" href="../css/journal.css">
</head>
<body>
    <header class="header">
        <nav class="nav">
            <div class="nav-brand">
                <a href="../index.html">🏠 我的空间</a>
            </div>
            <ul class="nav-links">
                <li><a href="../index.html">首页</a></li>
                <li><a href="../journal/index.html">日记</a></li>
                <li><a href="../timeline.html">时间线</a></li>
                <li><a href="../about.html">关于我</a></li>
            </ul>
        </nav>
    </header>

    <main class="container">
        <button class="back-btn" onclick="history.back()">← 返回日记列表</button>

        <article class="post-content">
            <div class="post-header">
                <h1>日记标题</h1>
                <div class="post-meta-large">
                    <span>📅 2026 年 03 月 26 日 星期四</span>
                    <div class="post-tags">
                        <span class="post-tag thoughts">思考</span>
                    </div>
                </div>
            </div>
            <div class="post-body">
                <!-- 日记正文 -->
            </div>
        </article>
    </main>
</body>
</html>
```
